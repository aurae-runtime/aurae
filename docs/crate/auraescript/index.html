An interpreted infrastructure language built for enterprise platform teams.

AuraeScript is an opinionated and Turing complete client language for an Aurae server. AuraeScript is an alternative to templated YAML for teams to express their applications.

The AuraeScript definition lives in this crate library (lib.rs).
